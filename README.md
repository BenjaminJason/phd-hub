# 美国 STEM 博士申请导航

面向中国学生的美国 STEM 博士（CS / EE / 物理 / 生物 / 数学等）申请单文件工具。

**在线访问：** https://benjaminjason.github.io/phd-hub/

## 功能
- 📅 申请时间轴（每步附「怎么做」指导）
- 🔬 96 位导师数据库（按方向 / 层次 / 学校筛选，含简介·邮箱·学生口碑）
- 🎯 选校匹配评分
- 📋 我的申请：套磁进度追踪 + 截止日期倒计时（数据存本地浏览器）
- ✉️ 套磁邮件模板 · ✅ 申请清单 · 💰 奖学金与申请费用预算
- 🛂 签证 & 政策 · 💬 学生说（25 校口碑）· 💼 就业 & 各城市生活
- 嵌入 PhD 日常 / SOP 视频、城市地图

> ⚠️ 数据为 **2026 年 6 月**整理的静态快照，导师招生状态 / GRE 政策 / 费用每年变动，**请以学校与导师官网为准**。
>
> 单文件 HTML，无需服务器。嵌入视频需在线访问才能播放（本地双击打开时视频无法加载，其余功能正常）。
